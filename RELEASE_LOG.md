## v1.3.0-rc.1 (2026-04-03)

### ✨ 已完成功能
1. **🧮 利润计算器功能**
   - 选品页面一键计算预估利润
   - 支持采购成本、运费、平台费率、其他成本自定义配置
   - 智能利润率评估和等级提示

2. **📦 选品库功能**
   - 独立MY LIBRARY页面，展示用户所有收藏商品
   - 支持按状态筛选（待比价/待生成文案/待发货/已上架）
   - 收藏功能对接Supabase数据库，数据持久化存储
   - 支持商品状态标签展示，不同状态颜色区分

3. **⚙️ 设置页面完善**
   - 默认市场配置：支持选择默认目标国家
   - 物流地址配置：支持填写发货地址、城市、邮编、国家
   - 选品偏好配置：默认平台费率、期望最低利润率
   - 配置本地持久化存储，刷新不丢失

### ✅ 灰度版本特性

## v1.2.0 (2026-04-02)
### 📌 版本信息
- **发布类型**: 正式版本
- **发布人**: SuperSolo Dev
- **Git标签**: `v1.2.0`
- **状态**: ✅ 已全量发布

### ✨ 核心更新内容
1. **🔥 选品功能全量上线**
   - 产品卡片组件，支持收藏、一键生成文案
   - 高级筛选组件，支持分类筛选、价格区间、多维度排序
   - 多国家市场切换适配

2. **🔌 第三方API对接框架完成**
   - 支持TikTok Shop、Amazon、Reddit多数据源
   - 自动降级机制，无凭证时自动使用Mock数据
   - 可通过环境变量一键切换数据源

3. **🧮 产品潜力评分算法优化**
   - 4维度加权评分：销售增长(40%)、社交热度(30%)、价格健康度(15%)、竞争度(15%)
   - 非线性评分提升区分度，更符合真实选品逻辑

4. **🤖 功能打通**
   - 选品数据与AI文案生成功能打通，支持从产品卡片一键生成高转化率营销文案

5. **🐛 Bug修复**
   - 修复所有生产构建错误，构建稳定性100%
   - 修复Next.js缓存损坏导致的UI样式丢失问题
   - 修复login页面useSearchParams()缺少Suspense边界问题
   - 修复API路由缺少dynamic配置导致的静态渲染错误

### 📋 影响范围
- 所有页面功能正常，无破坏性变更
- 对外API接口保持向下兼容

### 🔄 回滚方案
- 出现问题可直接回滚到上一个稳定版本 `v1.1.5`
- 回滚命令：`git reset --hard v1.1.5`
